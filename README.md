# Unicode Wave Function Collapse
Wave function collapse with unicode block string

## Usage
`*requires tqdm`
```bash
$ python uwfc.py --help
usage: all_test.py [-h] [--width WIDTH] [--height HEIGHT] [--n-iter N_ITER]

Create pattern with wave function collapse.

options:
  -h, --help       show this help message and exit
  --width WIDTH    width of generated pattern
  --height HEIGHT  height of generated pattern
  --n-iter N_ITER  how many patterns generates
```
